Exchange O' Learn - Documentación

Documentación de instalación y configuración de Exchange O' Learn.

Exchange O' Learn es una aplicación web SPA (Single Page Applicacion) diseñada especialmente para colegios e institutos, intalando esta aplicación en la red interna podemos aprovechar la velocidad de ésta para la transferencia eficaz de archivos. Maybe you want to have a single page documentation and use sidebar navigation for navigating inside the page. So this sample layouts are for you. Even maybe you'd like to use top navigation bar for navigating between your pages.

Introducción

While this page is enough to see through its code and get start with it, but you can see the page code structure in below. Make sure you'll have data-spy="scroll" data-target=".sidebar" data-offset="200" attribute in <body> tag.
